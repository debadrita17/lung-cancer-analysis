# Lung Cancer Data Analysis Project

## Project Overview
The goal of this project is to analyze lung cancer patient data to identify possible behavioral and demographic risk factors associated with lung cancer cases.

This project focuses on:
- Understanding patient demographics
- Overall cancer positivity trends
- Exploring smoking, Alcohol consumption and chronic disease patterns,impact
- Identifying high-risk age groups
- Creating interactive visual dashboards for insights

 ## Dataset Information

- Source: Kaggle
- File Type: CSV / Excel
- Dataset Type: Healthcare Dataset
- Total Records: 2,998 patients
- Tools Used:
  - Excel
  - Tableau

## Understanding the Dataset

Before starting data cleaning, the dataset was carefully examined to understand:

- Available columns and their meanings
- Types of data present in each column
- Important variables required for analysis
- Relationship between demographic and behavioral factors

This initial exploration helped in planning the analysis process properly.
 
 ## Data Cleaning Process

The dataset was cleaned and prepared using Excel.

Cleaning steps included:
- Checking missing values
- Standardizing column names
- Formatting categorical values
- Verifying data consistency
- Preparing data for visualization
The cleaned dataset was then used for further analysis and dashboard creation.

---

## Data Analysis Performed

The analysis was first performed using Excel tables and charts.

The following analyses were conducted:

- Total patient count analysis
- Gender-wise lung cancer distribution
- Age-wise patient analysis
- Risk factor comparison analysis like Smoking, Alcohol Consumption, Chronic Disease 
- Common symptoms analysis

The goal was to identify patterns and possible relationships between behavioral risk factors and lung cancer occurrence.

---

## Tableau Visualization & Dashboard

After completing the Excel analysis, Tableau was used to practice data visualization and dashboard creation.

The dashboard includes:

- Total patients KPI
- Average patient age
- Gender-wise cancer distribution
- Risk factor positivity analysis
- Age group analysis
- Summary insights section

The dashboard was designed to provide a clear visual understanding of lung cancer trends and risk factors.
